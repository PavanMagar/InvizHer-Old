Responsive Mini Portfolio Website Using HTML CSS and JavaScript

InvizHer

Project

12-04-2024

Good to see you here. As you know, today we are going to discuss about making of project "Responsive Mini Portfolio website using HTML CSS and JavaScript". This portfolio website is completely responsive with all devices. Also it is a completely ui/ux design based portfolio website. So without wasting too much time lets get start it to know about it.


Mini Portfolio Website

This is a simple and responsive portfolio website using HTML CSS and JavaScript. As it's name its a mini sized portfolio website, we used minimum sized components in it. The design of this portfolio website is little bit Ui and UX based. The overall look is just awesome. You can scroll down to watch it's live demo and source code.

Responsive Portfolio Website

This mini portfolio website is completely responsive with every device size. This means that whether a user is accessing the site from a desktop computer, a tablet, or a smartphone, the content and layout will adjust to provide an optimal viewing experience. Website will adjust size of its components automatically as per users device size.


Features of Mini Portfolio

  • It is a fast and speedy to load.
  • It is a responsive with different sized device. Supports almost every device.
  • Website has light mode and dark mode to switch with user friendly interface.
  • Developed first with the Mobile First methodology, then for desktop.
  • Compatible with all mobile devices and with a beautiful and pleasant user interface.
  • Website has user friendly interface. Design of website is little bit ui and ux based.
